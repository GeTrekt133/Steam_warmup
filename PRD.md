---

## 1. Обзор продукта

### 1.1 Описание
Десктопное приложение (Windows) для массового управления Steam-аккаунтами: авторегистрация, прогрев, фарм часов/кейсов CS2, управление инвентарём, повышение уровней и встроенный маркетплейс.

### 1.2 Целевая аудитория
- Фермеры Steam-аккаунтов (10–500+ аккаунтов)
- Трейдеры CS2 предметами
- Покупатели прогретых аккаунтов

### 1.3 Платформа
- Desktop (Windows 10/11), возможно cross-platform через Electron или PyQt
- Backend: Python (FastAPI) или Node.js
- Frontend: Electron + React / PyQt6 / Tauri

## 3. Нефункциональные требования

| Категория | Требование |
|-----------|-----------|
| Безопасность | Шифрование данных аккаунтов (AES-256) |
| Масштабируемость | Поддержка 500+ аккаунтов одновременно |
| Производительность | Параллельная обработка (asyncio / threading) |
| Надёжность | Логирование ошибок, retry-логика |
| UX | Тёмная тема, отзывчивый UI, прогресс-бары |
| Лицензирование | HWID-привязка или онлайн-активация |

## 4. Технический стек (Рекомендуемый)

| Компонент | Технология |
|-----------|-----------|
| Frontend | Electron + React + Tailwind CSS |
| Backend | Python FastAPI (локальный сервер) |
| Steam API | steam / steampy библиотеки |
| ASF | ArchiSteamFarm (embedded) |
| Captcha | hcaptcha-challenger + Claude/Gemini AI |
| БД | SQLite (локально) + PostgreSQL (облако) |
| Auth | JWT + HWID fingerprint |
| Прокси | aiohttp с SOCKS5/HTTP proxy |
| Шифрование | cryptography (Fernet/AES) |

```

│         Electron Frontend            │

               │ IPC / REST
┌──────────────▼──────────────────────┐

│  ┌─────────┐  ┌──────────────────┐  │

│  ┌─────────┐  ┌──────────────────┐  │

│  └─────────┘  └──────────────────┘  │
```

## 6. Метрики успеха

| Метрика | Цель |
|---------|------|
| Время регистрации 1 аккаунта | < 3 мин |
| Время выполнения бейджа | < 5 мин |
| Успешность captcha | > 85% |
| Стабильность (crash rate) | < 1% сессий |

## 7. Ограничения и риски

| Риск | Митигация |
|------|-----------|
| Бан аккаунтов Steam | Ротация прокси, имитация живого поведения |
| Изменение Steam API | Абстракция steam-клиента, быстрые обновления |
| Лимиты captcha-сервисов | Мультипровайдер (Claude, Gemini, 2captcha) |
| Утечка данных аккаунтов | AES-256 шифрование, опциональный offline-режим |
