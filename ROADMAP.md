# Roadmap — Steam Farming Panel
**Всего:** ~20 недель (5 месяцев)
**Команда:** 1-2 разработчика

---

## Фаза 0 — Фундамент (Недели 1–2)

**Цель:** Настроить проект, базовую архитектуру и UI-оболочку.

| Задача | Детали | Срок |
|--------|--------|------|
| 0.1 Выбор и настройка стека | Electron + React + Vite / или PyQt6 | День 1-2 |
| 0.2 Проект-шаблон | Базовая структура папок, linting, git | День 2-3 |
| 0.3 Навигация и Layout | Боковое меню, роутинг между вкладками | День 3-5 |
| 0.4 Тёмная тема + UI-компоненты | Таблицы, кнопки, модалки, progress-бары | День 5-10 |
| 0.5 Local Python Backend | FastAPI + SQLite, IPC-связь с frontend | День 5-10 |
| 0.6 Auth (локальный / облачный) | Логин, регистрация, HWID fingerprint | День 8-14 |

**Deliverable:** Запускаемое приложение с навигацией и пустыми вкладками.

---

## Фаза 1 — Ядро: Аккаунты и Прокси (Недели 3–4)

**Цель:** Основной функционал управления аккаунтами.

| Задача | Детали | Срок |
|--------|--------|------|
| 1.1 Парсинг TXT (login:pass) | Drag-and-drop + file picker | День 1-2 |
| 1.2 Импорт maFiles | Одиночный файл + папка | День 2-3 |
| 1.3 Таблица аккаунтов | Логин, email, прокси, статус, уровень | День 3-5 |
| 1.4 Модуль прокси | Добавление, проверка, round-robin привязка | День 5-8 |
| 1.5 Группы аккаунтов | CRUD групп, bulk-назначение | День 8-10 |
| 1.6 Шифрование данных | AES-256 через cryptography (Fernet) | День 8-10 |
| 1.7 Открыть в браузере | Auto-login через Playwright | День 10-14 |

**Deliverable:** Можно загрузить 500 аккаунтов, привязать прокси, сгруппировать.

---

## Фаза 2 — Регистрация аккаунтов (Недели 5–6)

**Цель:** Полностью рабочая авторегистрация Steam.

| Задача | Детали | Срок |
|--------|--------|------|
| 2.1 Интеграция текущего steamreg.py | Перенос/рефакторинг в архитектуру | День 1-3 |
| 2.2 Captcha pipeline | hcaptcha-challenger + Claude + fallback | День 3-7 |
| 2.3 SMS-верификация | Подключение SMS-активаторов (5sim, sms-activate) | День 5-8 |
| 2.4 Генерация профиля | Случайные имя, аватар, данные | День 7-9 |
| 2.5 Сохранение maFile | Автосохранение + добавление в аккаунты | День 9-12 |
| 2.6 UI регистрации | Прогресс, статус, лог | День 10-14 |

**Deliverable:** Зарегистрировать 10 аккаунтов в один запуск.

---

## Фаза 3 — ASF Интеграция (Недели 7–8)

**Цель:** Встроенный ASF без ручной конфигурации.

| Задача | Детали | Срок |
|--------|--------|------|
| 3.1 ASF subprocess менеджер | Запуск/стоп ASF как subprocess | День 1-3 |
| 3.2 Автогенерация bot-конфигов | Из аккаунтов панели → bot.json | День 2-4 |
| 3.3 Авторезолв 2FA | maFile → SteamGuard код | День 3-5 |
| 3.4 IPC с ASF | IPC-сервер ASF ↔ Python backend | День 5-9 |
| 3.5 Быстрые действия UI | Фарм часов/карточек, активация ключа | День 8-11 |
| 3.6 Мастер-аккаунт Family Sharing | Настройка через ASF команды | День 10-12 |
| 3.7 ASF Консоль | Raw terminal в UI | День 12-14 |

**Deliverable:** Создать 50 ASF-ботов в 2 клика, запустить фарм карточек.

## MVP (Минимально жизнеспособный продукт) — к концу Фазы 4

К 11-й неделе у вас будет:
- Импорт аккаунтов + прокси
- Авторегистрация
- ASF без конфигов
- Прогрев: бейджи + фарм часов

Этого достаточно для первых платящих пользователей.

---

## Стек итого

```
Frontend:  Electron + React 18 + Tailwind CSS + shadcn/ui
Backend:   Python 3.11 + FastAPI + SQLAlchemy + APScheduler
Steam:     steampy + aiosteam + Playwright
ASF:       ArchiSteamFarm (subprocess + IPC)
Captcha:   hcaptcha-challenger + Claude API + fallback
DB:        SQLite (local) + PostgreSQL (cloud sync)
Crypto:    cryptography (Fernet AES-256)
Build:     electron-builder (NSIS installer)
```
