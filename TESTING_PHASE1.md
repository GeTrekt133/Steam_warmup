# Сценарий ручного тестирования — Phase 1

Перед тестированием убедись, что запущены оба сервера:

```bash
# Terminal 1: Backend
cd Steam_warmup/backend && venv/Scripts/activate && python -m uvicorn app.main:app --reload --port 8420

# Terminal 2: Frontend
cd Steam_warmup/frontend && npm run dev
```

Открой http://localhost:5173

---

## 1. Регистрация

Открой http://localhost:5173 — увидишь форму логина. Нажми "Регистрация":

- Логин: `admin`
- Пароль: `123456`

## 2. Dashboard

После логина — главная. Должны быть 3 карточки с нулями (аккаунтов/прокси/групп: 0) и статус backend "OK".

## 3. Аккаунты — добавить вручную

Перейди в "Аккаунты" → кнопка "Добавить":

- Логин: `my_steam_acc`
- Пароль: `mypassword123`
- Заметка: `тестовый`

## 4. Аккаунты — импорт TXT

Кнопка "TXT" → вставь в текстовое поле:

```
steamuser1:password1
steamuser2:password2
steamuser3:password3
farmbot_01:botpass123
farmbot_02:botpass456
```

Нажми "Импортировать" — должно показать "Импортировано: 5, пропущено: 0".

## 5. Аккаунты — импорт maFile

Кнопка "maFile" → вставь этот JSON:

```json
{
  "account_name": "steamuser_guard",
  "SteamID": "76561198999999999",
  "shared_secret": "wGJfMHAHEh0DTYV8EFZfTdkGUbE=",
  "identity_secret": "pN0+Qe3KxXLGOCmaalGMOuPdJ7k=",
  "Session": {
    "SteamID": "76561198999999999"
  }
}
```

Пароль: `guardpass123`

Этот аккаунт появится с меткой "2FA" в столбце maFile.

## 6. Steam Guard код

У аккаунта `steamuser_guard` в столбце действий будет иконка щита — нажми. Увидишь 5-символьный код (типа `V2K9R`) с таймером TTL. Кнопка "Копировать" скопирует в буфер, "Обновить" — сгенерит новый.

## 7. Поиск и фильтрация

В поле поиска набери `farm` — останутся только `farmbot_01` и `farmbot_02`. Очисти поиск.

## 8. Выделение + массовое удаление

Поставь галочки на `farmbot_01` и `farmbot_02` → появится красная кнопка "Удалить (2)" → нажми → подтверди.

## 9. Прокси — добавить

Перейди в "Прокси" → "Добавить":

- Хост: `192.168.1.100`
- Порт: `8080`
- Протокол: HTTP

## 10. Прокси — импорт TXT

Кнопка "Импорт TXT" → вставь:

```
103.152.112.2:8080
socks5://45.67.89.10:1080
185.199.228.100:3128:proxyuser:proxypass
http://admin:secret@91.234.56.78:3128
```

Должно импортировать 4 штуки.

## 11. Проверка прокси

Кнопка "Проверить все" — будет крутиться спиннер. Все прокси покажут красный крестик (они фейковые, недоступны) — это нормально. Пинг покажет "—".

## 12. Открыть в браузере

Вернись в Аккаунты → нажми иконку глобуса на любом аккаунте. Бэкенд запустит Playwright Chromium с автологином (с фейковым аккаунтом увидишь ошибку Steam — это ок, главное что браузер открылся).

---

## Форматы данных

| Тип | Формат |
|-----|--------|
| TXT аккаунты | `login:password` — по одной паре на строку |
| maFile | JSON от SteamDesktopAuthenticator (обязательные поля: `account_name`, `shared_secret`, `identity_secret`) |
| TXT прокси | `host:port`, `host:port:user:pass`, `protocol://host:port`, `protocol://user:pass@host:port` |

Все данные выше — фейковые, но формат соответствует реальному.
